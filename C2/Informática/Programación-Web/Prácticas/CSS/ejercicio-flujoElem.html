EJERCICIO FLUJO DE ELEMENTOS

Se emplea para ubicar elementos según un diseño del documento que hayamos hecho. Si no se indica lo contrario, los elementos se irán apilando como bloques unos a continuación de otros. Para poder cambiar ese diseño se emplea el atributo float, el cual toma como atributos left o right. Así, al indicarle que se vaya a la izquierda, o a la derecha, el elemento se moverá a uno de esos lados tanto como pueda.
En este caso, las imágenes irán todas a la derecha y si hay texto, este quedará a su izquierda. Para volver al diseño por defecto (bloques unos a continuación de otros), se emplea el atributo clear, el cual puede tomar los valores left, right, both, none e inherit, indicando qué justificación es la que se libera.
